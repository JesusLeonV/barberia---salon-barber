# The Salon Barber 💈

Sistema web moderno y responsivo para la gestión de reservas de servicios de barbería en tiempo real, diseñado con un enfoque oscuro, elegante e intuitivo para el usuario.

## 🚀 Características clave
* **Buscador interactivo:** Filtro rápido de servicios en tiempo real.
* **Componentes dinámicos:** Secciones colapsables (accordions) para organizar categorías como Cortes, Barba y Combos.
* **Calendario de citas:** Selector visual de días y horas optimizado para agendamiento fluido.
* **Diseño 100% Responsivo:** Adaptado perfectamente para computadoras, tablets y teléfonos móviles.
* **Footer Profesional:** Incluye información de contacto, horarios y enlaces de políticas legales esenciales.

## 🛠️ Tecnologías utilizadas
* **HTML5:** Estructura semántica avanzada.
* **CSS3:** Estilos personalizados, Flexbox, Grid y animaciones de transición suaves (`fadeUp`).
* **JavaScript (Vanilla):** Lógica para el manejo de estados en el calendario, colapsables de servicios y navegación dinámica.
